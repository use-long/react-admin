##### 描述

修改权限列表

#### 请求URL

`/brand/updateBrand`

#### 请求方式

####   post

#### 参数

| 参数名 | 必选 | 类型    | 说明                         |
| ------ | ---- | ------- | ---------------------------- |
| id     | 是   | int     | 品牌标识，作为参数查找到品牌 |
| name   | 是   | varchar | 修改的品牌名                 |
| image  | 是   | varchar | 修改的图片地址               |
| letter | 是   | char    | 修改的首字母                 |
