## 添加优惠券

#### 请求URL

· `/addcoupons/getAddCoupons`

#### 请求方式

- POST

#### 参数

|参数名   | 是否必选
|store_id |  是
|spu_id   |  是
|start_time |  是
|end_time |  是
|deno     |  是
|condition |  是
|name |  是
